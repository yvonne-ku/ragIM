yep Although, if you're running DHCP it will eventually change (which is what makes this lousy) If you assigned static addresses, you are NOT using dhcp.  If you plug in your computer and it works, you are using DHCP. Often, on a small network, the DHCP lease ends, the client requests a new IP, and the router gives it the same IP.  This happens over and over so sometimes the DHCP assigned address appears to never change. But usually the lease is something like a day or a week.

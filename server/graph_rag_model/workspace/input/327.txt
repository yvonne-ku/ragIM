To file a Civil Harassment Restraining Order in Sacramento Superior Court, you need to complete the Request for Civil Harassment Restraining Order (Step 1) Packet (CV\E-CH-01) and present the completed forms to the Civil Front Counter - Window 14 in Room 102 of the Gordon D.

What about Hebrew? Hebrew is not included in the supported languages for developing dialog skills in Watson Assistant. However, you can utilize the universal language model to create a conversation in a language for which no dedicated language model is available. Can it handle the reverse order of text? Yes, the universal language model can handle the reverse order of text if the skill supports a language with right-to-left directional text, such as Hebrew.
